IQTell to Nozbe migration tool

Developers note:

I loved using IQTell but now it will shutdown, it is time to move on. For me Nozbe is very good alternative. It has a great user interface and great apps. I created this tool for fellow IQTell users who want to migrate to Nozbe.

@Ran: Thanks for the ride. Wish you well.

Ben Mens




Using this tool you can import your IQTell data into Nozbe. Just export your IQTell activities and IQTell projects to your disk and then upload the files using the wizzard below. You can use thit tool multiple times. It will merge the imported data with the existing data from Nozbe.
